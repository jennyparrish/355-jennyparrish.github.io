Code Log 4: Colors with CSS

Once we have our basic HTML framework, we can begin customizing our site in a variety of ways, usually beginning with color. It is possible to add color directly to the HTML file, which is done within the area and enclosed within the section. Often it is preferable (like when your CSS is more complex) to create a style.css file to be accessed by the HTML file directly. Using CSS colors, we can create an infinite variety of palettes, feelings, and effects within our websites that inhabit whatever vibe we choose. CSS colors give us access to nearly every shade imagineable, and they do this using four distinct methods:

  • Named Colors: These are predefined color names like "red", "blue", "green", etc. They are easy to remember and use, but they offer a limited palette.
  • Hexadecimal Colors: These use a six-digit code to represent colors, starting with a "#" symbol. For example, "#FF5733" represents a bright orange color. Hex codes allow for a much wider range of colors than named colors.
  • RGB Colors: This method uses the RGB (Red, Green, Blue) color model to define colors. You can specify the intensity of each color component using values from 0 to 255. For example, "rgb(255, 87, 51)" would produce the same bright orange as the hex code above.
  • HSL Colors: HSL stands for Hue, Saturation, and Lightness. This method allows you to define colors based on their hue (the type of color), saturation (the intensity of the color), and lightness (how light or dark the color is). For example, "hsl(14, 100%, 60%)" would also produce the same bright orange.

By mastering these color methods, we can create visually appealing and cohesive designs that enhance the user experience and effectively communicate our agenda. Whether we want to evoke a specific emotion, create a brand identity, or simply make our website more attractive, understanding and utilizing CSS colors is essential for any web developer.

The names of the most commonly used color attributes are:

  • "color"
  • "background-color", and
  • "border-color"

The CSS "color" property can change the foreground color of an elements' text content, including certain elements like underlines or focus indicators.

The"background-color" property fills the area behind an element's content, but it's extent is defined by the element's content box plus padding.

Using the "border-color" property, we can draw a line around whatever tag it is assigned to; although, we must also designate border-style and border-width for the border to show-up. Borders can be placed around all different kinds of elements, from text, to images, to entire sections. They can be made in any color, and can be thick, thin, dotted, dashed, or even invisible.
