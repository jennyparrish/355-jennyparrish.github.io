Code Log 2: CSS Measuring Units

Introduction

In this code log, I will explore various CSS measuring units and their applications in web design. Understanding these units is crucial for creating responsive and visually appealing layouts.

Absolute Units

Absolute units include pixels (px), points (pt), inches (in), centimeters (cm), and millimeters (mm). These units are fixed and do not scale based on the viewport size.

Relative Units

Relative units include percentages (%), em, rem, vw, vh, vmin, and vmax. These units scale based on the parent element or the viewport size, making them ideal for responsive design.

  • em: Relative to the font-size of the parent element.
  • rem: Relative to the font-size of the root element.
  • vw: Relative to 1% of the viewport's width.
  • vh: Relative to 1% of the viewport's height.
  • vmin: Relative to 1% of the smaller dimension of the viewport.
  • vmax: Relative to 1% of the larger dimension of the viewport.
